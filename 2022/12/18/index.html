2022/12/18

html

That you can group fields in an html form with a <fieldset> tag, and that it accepts a top-level child <legend> tag. Together,

<fieldset>
  <legend>Legend</legend>
  <div>...</div>
</fieldset>

produces

  ┌────legend─────┐
  │               │
  │               │
  │      ...      │
  │               │
  │               │
  └───────────────┘

which is simultaneously cool and hard to style.
